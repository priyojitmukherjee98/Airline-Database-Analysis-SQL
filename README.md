# Airline Customer & Flight Data Analysis

## 📌 Overview
This project analyzes airline customer and flight data to uncover patterns in bookings and travel behavior using SQL.

## 🎯 Objective
To analyze:
- Customer booking patterns
- Flight activity
- Travel behavior trends

## 🛠️ Tools Used
- SQL (MySQL / PostgreSQL)

## 🔍 Key Steps
- Queried relational datasets
- Analyzed booking frequency
- Explored customer-flight relationships
- Identified operational patterns

## 📊 Key Insights
- Customer travel frequency trends
- Popular routes and booking behaviors
- Operational insights from flight data

## 📈 Outcome
Generated structured insights to support data-driven decision-making in airline operations.
